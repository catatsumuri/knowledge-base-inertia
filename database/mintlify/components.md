# 概要

> レイアウト、強調、API ドキュメント、ナビゲーションのためのコンポーネントライブラリ。

Mintlify は、ドキュメントページ用の組み込み MDX コンポーネントを提供します。これらのコンポーネントを使って、コンテンツを構造化し、重要な情報に注意を向けさせ、API を文書化し、ナビゲーションを案内できます。
