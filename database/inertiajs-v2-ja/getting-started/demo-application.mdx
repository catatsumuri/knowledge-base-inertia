title: デモアプリケーション
---

Inertia.js 用のデモアプリ [Ping CRM](https://demo.inertiajs.com) をセットアップしました。このアプリケーションは Laravel と Vue を使用して構築されています。ソースコードは [GitHub](https://github.com/inertiajs/pingcrm) で確認できます。

Ping CRM のデモは Heroku 上でホストされており、データベースは毎時間リセットされます。データを編集する際はご配慮ください。

![Ping CRM](https://mintcdn.com/inertiajs/Rp49etZkBLard8yO/images/pingcrm.png?w=2500&fit=max&auto=format&n=Rp49etZkBLard8yO&q=85&s=8c2ef080c2f1dc5844eee696f6c2c1cd)
*https://demo.inertiajs.com*

Ping CRM の Vue 版に加えて、Svelte 版のアプリケーションも管理しています。こちらは [GitHub](https://github.com/inertiajs/pingcrm-svelte) で確認できます。
